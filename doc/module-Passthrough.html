Module: Passthrough

This file contains functions meant to consolidate all EarSketch API functions into a single source. Both the Python and Javascript libraries are simply wrappers for these functions.

If your API function needs to be asynchronous, make sure it returns a promise, and use suspendPassthrough() in the Javascript and Python wrappers.
