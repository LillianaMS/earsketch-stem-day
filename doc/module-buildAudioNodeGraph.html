(inner) staircaseRamp(callback, interval, sync)

Linear parameter slope function for the effects that doesn't support linearRampAtTime(). The first and last values are always updated at the user-set location. This function is to be used within the paramter updates for the upcoming ramps. See 'REVERB_TIME' implementation for the example.

Parameters:
Name Type Description
callback function

Custom function to update to the current parameter. It receives the current relative-to-location effect value as the first argument.
interval number

Interval in location unit. E.g. 0.25 = 1 quarter note.

Syncs the update points to the nearest beats. Currently unavailable.
