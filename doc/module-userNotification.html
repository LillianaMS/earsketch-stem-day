Module: userNotification

Angular directive / service for showing user notifications. Inject userNotification to Angular submodules, and put in HTML. Currently only 1 instance of can be used. The notification service is copied to the global for non-angular usage.

Parameters:
Name Type Attributes Default Description
text string

Notification message to show
type string <optional>
 'normal'

Sets color style for different message types. Supported: 'normal' (default), 'success', 'failure1', 'failure2'
duration number <optional>
 2.5

Duration in seconds for displaying the message.
Source:
  • app/app.js, line 48
Examples
userNotification.show('Hello user!');
userNotification.show('Oops, something broke!', 'failure1');
userNotification.show('Show this message for a long time', null, 10);
userNotification.hide(); // hide at any time.
